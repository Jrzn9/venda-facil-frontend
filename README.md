# Venda Fácil - Sistema de Cadastro de Veículos

Este projeto consiste no desenvolvimento de uma interface front-end para um sistema de anúncios de veículos. A aplicação foca na construção de um formulário de cadastro estruturado, garantindo uma experiência de usuário fluida e uma coleta de dados eficiente.

## Descrição do Projeto

O objetivo deste repositório é apresentar uma solução de interface para usuários que pretendem anunciar veículos de forma rápida. O sistema valida as informações inseridas e permite a inclusão de detalhes técnicos essenciais para o mercado automotivo.

## Funcionalidades Principais

* **Formulário de Registro:** Campos específicos para título do anúncio, preço, marca, modelo e quilometragem.
* **Validação de Dados:** Utilização de atributos HTML5 para garantir o preenchimento de campos obrigatórios e formatos corretos.
* **Seleção de Atributos:** Componentes para escolha de tipo de transmissão (Manual ou Automática) e data de aquisição.
* **Upload de Arquivos:** Campo configurado para aceitar múltiplos arquivos de imagem.

## Tecnologias Utilizadas

* **HTML5:** Estruturação semântica de elementos e formulários.
* **CSS3:** Estilização personalizada, gestão de layout e efeitos de interação.

## Estrutura do Repositório

* `index.html`: Arquivo principal contendo a estrutura da página.
* `css/style.css`: Folha de estilo responsável pelo design e interface visual.
